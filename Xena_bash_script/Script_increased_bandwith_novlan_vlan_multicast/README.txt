1. Start Cygwin terminal
2. Open the file xena_main.sh
3. In the code:
	3.1 Set protNo
	3.2 Choose function 
	protNo = 4/4, choose the function Novlan_random_bandwidth200M_1G --> Port 4/4 produces the stream without VLAN
	protNo = 4/5, choose the function Vlan_random_bandwidth200M_1G   --> Port 4/5 produces the stream without VLAN
4. run ./xena_main.sh at the Cygwin terminal
